# TreeView Web App

Next.js app for TreeView.

Contains both UI pages and API (all server-side).

This package uses:

- [`packages/web-ui`](../../packages/web-ui): UI library
- [`packages/trpc`](../../packages/trpc): tRPC routers and configs

## Setup

```bash
pnpm install
```

## Available scripts

### Run app

```bash
pnpm dev
pnpm start # server
```

### Build app

```bash
pnpm build
```

### Lint app

```bash
pnpm lint

# To fix lint
pnpm lint:fix
```
