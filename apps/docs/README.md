# TreeView documentation Web App

Next.js app for TreeView's documentation

## Setup

```bash
pnpm install
```

## Available scripts

### Run apps

```bash
pnpm run dev
pnpm run start # server
```

### Build apps

```bash
pnpm run build
```

### Lint apps

```bash
pnpm run lint

# To fix lint
pnpm run lint:fix
```
