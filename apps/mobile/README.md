# TreeView Mobile App

Expo app for TreeView.

## Setup

```bash
pnpm install
```

## Available scripts

### Run app

```bash
pnpm dev

# For device-specific runs
pnpm dev:android
pnpm dev:ios
```

### Build app

```bash
pnpm build
```

### Lint app

```bash
pnpm lint

# To fix lint
pnpm lint:fix
```
