# TreeView Database

MySQL database accessible using Prisma.

## Setup

```bash
pnpm install
```

## Available scripts

### Generate Prisma client

```bash
pnpm -F database generate
```

### Seed example data

```bash
pnpm -F database db:seed
```

### Lint apps

```bash
pnpm lint
# To fix lint
pnpm lint:fix
```
