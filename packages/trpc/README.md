# TreeView tRPC routers and configs

tRPC routes and context definitions for TreeView apps.

## Setup

```bash
pnpm install
```

## Available scripts

### Run app

```bash
pnpm dev
pnpm start # server
```

### Build app

```bash
pnpm build
```

### Lint app

```bash
pnpm lint

# To fix lint
pnpm lint:fix
```
